Веб-приложение для управления задачами команды, разработанное в рамках учебного проекта.

## 🎯 Цель проекта

Получить практические навыки командной разработки программного обеспечения с использованием Git, GitHub и базовых веб-технологий.

## ✨ Функциональность

- ✅ Добавление новых задач с указанием исполнителя
- ✅ Отметка задач как выполненных
- ✅ Удаление задач
- ✅ Фильтрация задач (все/активные/выполненные)
- ✅ Счётчик общего количества задач и активных задач

## 🛠️ Технологии

- CSS3
- JavaScript (Vanilla JS)
- Git & GitHub
