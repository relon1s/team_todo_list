## Тест-кейсы

### Тест 1: Добавление задачи
**Шаги:**
1. Ввести текст задачи: "Изучить Git"
2. Ввести исполнителя: "Иван"
3. Нажать кнопку "Добавить"

**Ожидаемый результат:**
- Задача появляется в списке
- Поля ввода очищаются
- Счётчик обновляется

**Статус:** ⬜ Passed / ⬜ Failed

---

### Тест 2: Добавление задачи без текста
**Шаги:**
1. Оставить поле задачи пустым
2. Ввести исполнителя
3. Нажать "Добавить"

**Ожидаемый результат:**
- Появляется сообщение "Введите текст задачи!"
- Задача не добавляется

**Статус:** ⬜ Passed / ⬜ Failed

---

### Тест 3: Отметка задачи как выполненной
**Шаги:**
1. Добавить задачу
2. Нажать на чекбокс рядом с задачей

**Ожидаемый результат:**
- Текст задачи зачёркивается
- Задача становится полупрозрачной
- Счётчик активных задач уменьшается

**Статус:** ⬜ Passed / ⬜ Failed

---

### Тест 4: Удаление задачи
**Шаги:**
1. Добавить задачу
2. Нажать кнопку "Удалить"
3. Подтвердить удаление

**Ожидаемый результат:**
- Задача исчезает из списка
- Счётчик обновляется

**Статус:** ⬜ Passed / ⬜ Failed

---

### Тест 5: Фильтр "Активные"
**Шаги:**
1. Добавить 3 задачи
2. Отметить 1 задачу как выполненную
3. Нажать кнопку "Активные"

**Ожидаемый результат:**
- Отображаются только невыполненные задачи (2 штуки)

**Статус:** ⬜ Passed / ⬜ Failed

---

### Тест 6: Фильтр "Выполненные"
**Шаги:**
1. Нажать кнопку "Выполненные"

**Ожидаемый результат:**
- Отображается только 1 выполненная задача

**Статус:** ⬜ Passed / ⬜ Failed
